# Getting Started

In VS Code, use [Live Server by Ritwick Dey](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) to run the `.html` file.

Check within `sketch.js` to adjust settings like:

-   Speed
-   Jump height
-   Game score
-   And more
